## Oportunidades AI en LATAM

1. **Migración a AI coding tools en enterprises LATAM**: La mayoría de empresas medianas en Brasil, México y Colombia aún usan Copilot básico o no usan AI. Oportunidad de desplegar stacks completos (Dify + OpenCode + n8n) con soporte local.

2. **Self-hosted AI DevOps**: Regulaciones de datos en México, Brasil (LGPD) y Argentina impulsan demanda de soluciones AI 100% on-premise. Stack: Ollama + OpenCode + Gitea + MLflow.

3. **AI-native CI/CD para fintechs LATAM**: Las fintechs latinoamericanas necesitan testing automatizado con AI + security review continuo. Claude Code Security Review Action + LangGraph workflows.

4. **Formación y productividad para dev shops**: Miles de software factories en LATAM pueden duplicar productividad con AI coding agents. Oportunidad de training + setup + managed services.

## Posicionamiento Globant

- **Fortaleza**: Globant tiene +27k ingenieros y relaciones con enterprise clients que pueden adoptar AI coding tools a escala
- **Diferenciador**: Capacidad de configurar stacks open source (sin vendor lock) + expertise en integración con sistemas legacy
- **Opportunity**: AI-native software delivery — ofrecer a clientes reducción 30-60% en tiempo de desarrollo usando agentes open source curados
- **Risk**: Necesita mover rápido — la adopción de Claude Code/Cursor en ingenieros individuales está superando a los programas enterprise
