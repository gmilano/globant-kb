## This Week's Breakout

**Mastra** is the fastest-rising new framework, gaining ~4k stars/month in mid-2026. It fills the TypeScript gap that LangChain (Python-first) left open, with native Vercel/Cloudflare Edge deployment and a built-in workflow engine.

**OpenHands Software Agent SDK** (newly released 2026) is the most exciting new repo for Globant: it's the buildable SDK layer on top of the OpenHands runtime, enabling branded coding agent products.
