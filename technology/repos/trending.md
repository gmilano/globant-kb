## Signals to Watch

- **MCP server registry**: [wong2/awesome-mcp-servers](https://github.com/wong2/awesome-mcp-servers) — community registry growing fast; hundreds of integrations (GitHub, Jira, Slack, AWS, K8s, Postgres). The npm-equivalent for AI tool connectivity is forming.
- **Coding agent consolidation**: With Roo Code archived, market consolidating: Cline (VS Code), Aider (terminal), OpenHands (autonomous), OpenCode (terminal). Watch for ZooCode (Roo Code community fork).
- **LLM observability**: Langfuse (MIT, 12k+), Phoenix (Arize), SigNoz all adding LLM/agent-specific traces — now a procurement requirement for enterprise agent deployments.
- **CNCF AI momentum**: Backstage (2026 documentary), Prometheus, ArgoCD, OpenTelemetry — cloud-native standards becoming the backbone for AI-native dev platforms.
- **n8n crossing 182k**: Workflow automation as the "boring AI work" entry point — every enterprise wants Zapier with LLM nodes. n8n (source-available ELv2) is the go-to open option.
- **Hermes Agent breakout**: 188k stars is the fastest sustained GitHub growth in agentic AI history. The self-improving skills model is the new design pattern to watch; expect widespread imitation in H2 2026.
- **SWE-bench milestone**: 80%+ accuracy by top models (Claude Mythos 93.9%, Opus 4.8 88.6%) — AI coding agents have crossed the "senior engineer" threshold on standardized tasks. Every AI coding product pitch now leads with a SWE-bench number.
- **MLOps + LLMOps convergence**: MLflow 3.0 LLM tracing means the same tool that tracks your training experiments now tracks your production LLM calls. Kubeflow adding native LLM pipeline support. The MLOps and LLMOps categories are merging into "AI engineering platforms."
