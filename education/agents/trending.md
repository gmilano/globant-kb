# Trending AI Agents — Education (Week of 2026-07-02)

> What's new, what's spiking, what just shipped

## Breakout Projects This Week

### 1. DeepTutor v1.0 (Complete Rewrite) — Apache 2.0
**Repo**: [HKUDS/DeepTutor](https://github.com/HKUDS/DeepTutor)
**Stars**: 23k+ (was 1.4k at launch in Dec 2025; rewrote entirely in April 2026)

DeepTutor v1.0 shipped as a ground-up rewrite — modular multi-agent architecture, persistent learner memory across sessions, MCP (Model Context Protocol) compatibility, and first-class support for self-hosting with Ollama. The shift from "AI reading assistant" to "agent-native tutoring workspace" is what triggered the star explosion. Now integrates with 30+ LLM providers and ships with a Docker Compose one-liner.

**Why it matters for Globant**: This is the reference architecture for AI tutoring agents in 2026 — study it before any education engagement.

---

### 2. GitHub Copilot Student Plan Launch (March 2026)
GitHub launched a dedicated Copilot Student plan (replacing ad-hoc complimentary access), bundling it into the GitHub Education pack. 20M+ students now have enterprise-grade AI code assistance. This is accelerating adoption of AI-assisted coding in CS education and creating demand for AI-literacy curriculum.

**Opportunity**: Clients in CS education need updated curricula that teach students to work with, not against, AI coding assistants.

---

### 3. OECD Digital Education Outlook 2026
The OECD released its 2026 Digital Education Outlook recommending a pivot from **generic AI tools** (ChatGPT for homework) to **purpose-built educational AI** that produces measurable learning gains. This is driving procurement decisions at K-12 districts and universities globally.

**Impact**: Strong tailwind for custom-built, outcome-measured educational AI platforms over generic chatbot deployments.

---

### 4. Moodle AI Manager Plugin (local_ai_manager)
**Plugin**: [moodle.org/plugins/local_ai_manager](https://moodle.org/plugins/local_ai_manager)

New multi-tenant AI management plugin for Moodle — allows different departments/courses to connect different AI backends. Supports OpenAI-compatible APIs, enabling substitution with self-hosted models. Becoming the standard AI integration layer for Moodle deployments.

---

### 5. Microsoft × Khan Academy Phi-3 Open-Source Math Tutoring
Microsoft and Khan Academy announced an open-source small language model (SLM) based on Phi-3 for math tutoring — designed to run on low-resource devices and be freely redistributable. Targets underserved student populations globally.

**Repo**: Watch [microsoft/Phi-3](https://github.com/microsoft/Phi-3CookBook) for the education-specific fine-tunes.
