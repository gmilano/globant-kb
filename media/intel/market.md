## Verticales más atractivas para Globant AI Studios

1. **Localización y subtitulación a escala** — combinar Whisper + traducción LLM + VideoCaptioner para streamers y broadcasters.
2. **Post-producción agentic** — OpenMontage + SkyReels V2 + AudioCraft para reducir costos en el pipeline editorial.
3. **Content Intelligence** — media_monitor + Claude para análisis de audiencia, trending topics y decisiones editoriales.
4. **Video generativo para marketing** — Wan2.2 + ComfyUI para agencias y marcas que necesitan video at scale.
5. **Radio/Podcast AI** — AudioCraft + Bark + Jellyfin/Icecast para crear emisoras automatizadas o programas AI-assisted.

## Posicionamiento Globant

- **Diferenciador**: capacidad de integrar modelos open source (Apache/MIT) con infraestructura enterprise y cumplimiento (GDPR, LGPD).
- **Sweet spot LATAM**: localización multilingüe AI para España, Brasil, México, Colombia.
- **Vertical de entrada**: post-producción (subtítulos + transcripción + metadatos) — TCO muy claro para broadcasters.
- **Expansión**: content generation pipeline para marcas y agencias digitales.
