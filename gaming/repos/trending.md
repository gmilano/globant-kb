## Señales de mercado a seguir

- **AlayaWorld** (Shanda AI, jul 2026): world model open-source para gaming con 60s+ coherente. Primer candidato viable para AI-native games.
- **GamingAgent** + **GRL** (lmgame-org, ICLR 2026, MIT): pipeline completo de evaluación + mejora de LLMs para juegos via RL.
- **COS-PLAY** (MIT): patrón co-evolving para agentes que aprenden habilidades durante el juego.
- **Carbon Engine** (MIT): 23 años de código EVE Online, modular → Destiny pathfinding o Trinity graphics aprovechables.
- **7,300 juegos Steam** declaran AI en 2026 (2× 2024): mainstream irresistible.
- **Unity AI Inference Engine**: LLMs/modelos ML en runtime Unity sin servidor. Tendencia hacia AI on-device.
- **Godot Foundation AI ban** (1-jul-2026): PRs con AI en engine core prohibidos. Plugins/addons sin restricción.
