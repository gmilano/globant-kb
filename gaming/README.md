## Quick Reference

### Must-Know Repos
- **Engine:** [godotengine/godot](https://github.com/godotengine/godot) — MIT, 90k+ stars
- **RL for NPCs:** [Unity-Technologies/ml-agents](https://github.com/Unity-Technologies/ml-agents) — Apache 2.0
- **Local LLM NPCs:** [Adriankhl/godot-llm](https://github.com/Adriankhl/godot-llm) — MIT
- **Game backend:** [heroiclabs/nakama](https://github.com/heroiclabs/nakama) — Apache 2.0

### Top Composition Patterns
1. Offline AI NPC (Godot + godot-llm + NobodyWho)
2. RL Playtesting Bot (ML-Agents or Godot RL Agents)
3. AI Matchmaking (Nakama + Open Match + ML model)
4. Asset Pipeline (InvokeAI + FLUX.1 LoRA + ComfyUI)
5. Studio Copilot (MetaGPT or OpenHands + Godot AI module)

### Key Insight for Clients
AI could unlock **$22B** in profit for gaming companies by cutting production costs 30–45% and enabling micro-studios of 3–5 people to produce AAA-quality games. The open-source stack (Godot + local LLMs + Nakama) makes this achievable without per-seat licensing fees.
