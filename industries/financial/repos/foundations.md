# Foundational Repos — Financial AI

_These are the battle-tested open source libraries that underpin financial AI work._

| Repo | License | Stars | Language | Description |
|------|---------|-------|----------|-------------|
| [AI4Finance-Foundation/FinRL](https://github.com/AI4Finance-Foundation/FinRL) | MIT | 10K+ | Python | Financial reinforcement learning — standard framework for training trading agents; multi-asset, multi-environment |
| [AI4Finance-Foundation/FinGPT](https://github.com/AI4Finance-Foundation/FinGPT) | Apache 2.0 | 14K+ | Python | Financial LLMs with fine-tuning pipeline; sentiment, robo-advisor, forecasting, chatbot use cases |
| [AI4Finance-Foundation/FinRobot](https://github.com/AI4Finance-Foundation/FinRobot) | Apache 2.0 | 3K+ | Python | Multi-agent orchestration platform for financial analysis using LLMs |
| [virattt/ai-hedge-fund](https://github.com/virattt/ai-hedge-fund) | MIT | 50K+ | Python | Multi-agent investment committee — the most-starred AI finance project; great starting point for demos |
| [OpenBB-finance/OpenBB](https://github.com/OpenBB-finance/OpenBB) | AGPL-3.0 | 32K+ | Python | Financial data platform replacing Bloomberg Terminal workflows; extensible with custom agents |
| [freqtrade/freqtrade](https://github.com/freqtrade/freqtrade) | GPL-3.0 | 30K+ | Python | Production-grade crypto trading bot with backtesting, ML strategy optimization |
| [lballabio/QuantLib](https://github.com/lballabio/QuantLib) | BSD-3-Clause | 5K+ | C++ | 30-year-old quantitative finance library: derivatives pricing, yield curves, risk measures — industry standard |
| [stefan-jansen/zipline-reloaded](https://github.com/stefan-jansen/zipline-reloaded) | Apache 2.0 | 2K+ | Python | Event-driven equity backtesting (used in ML for Algorithmic Trading textbook) |
| [georgezouq/awesome-ai-in-finance](https://github.com/georgezouq/awesome-ai-in-finance) | MIT | 9K+ | — | Curated list of LLMs, deep learning, and quant tools for financial markets |
| [wilsonfreitas/awesome-quant](https://github.com/wilsonfreitas/awesome-quant) | MIT | 15K+ | — | Comprehensive curated list of quant finance libraries, datasets, and tools by language |
