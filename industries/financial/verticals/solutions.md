# Vertical Solutions — Financial Industry

_Open source platforms that can be customized with AI on top. Prioritized by Globant-usable licenses._

## Core Banking & Microfinance

### [Apache Fineract](https://github.com/apache/fineract)
- **License:** Apache 2.0
- **Language:** Java (Spring Boot)
- **Domain:** Core Banking Platform
- **Description:** Full-stack microfinance platform powering financial services for 2 billion+ underbanked globally. Features: client management, loan portfolio, savings accounts, real-time accounting, payment gateway integrations, reporting.
- **AI Opportunities:**
  - LLM agent for loan officer decision support
  - Fraud detection layer on transaction streams
  - NLP for KYC document processing
  - Conversational interface via Rasa
- **Used by:** Mifos, FINERACT-CN deployments in 30+ countries

## Accounting & ERP

### [Akaunting](https://github.com/akaunting/akaunting)
- **License:** GPL-3.0
- **Language:** PHP (Laravel)
- **Domain:** Accounting / SME Finance
- **Description:** Modern online accounting software for SMEs and freelancers. Core features: invoicing, expense tracking, income management, asset tracking, budgeting, inventory, multi-currency.
- **AI Opportunities:**
  - LLM-based invoice classification and anomaly detection
  - Cash flow forecasting with FinGPT-style sentiment layer
  - Natural language query interface for accountants

### [iDURAR ERP-CRM](https://github.com/idurar/idurar-erp-crm)
- **License:** MIT
- **Language:** Node.js (Express) + React + MongoDB
- **Domain:** ERP/CRM/Accounting
- **Description:** All-in-one ERP-CRM for SMEs: customer management, sales, procurement, inventory, financials, invoicing in a unified interface.
- **AI Opportunities:**
  - Sales forecasting agents
  - Automated procurement recommendation
  - Financial KPI dashboard with LLM narrative

### [Apache OFBiz](https://github.com/apache/ofbiz-framework)
- **License:** Apache 2.0
- **Language:** Java + Groovy
- **Domain:** Full ERP
- **Description:** Mature Apache Foundation ERP covering accounting, manufacturing, supply chain, CRM, HR, and e-commerce.
- **AI Opportunities:**
  - AI-powered demand forecasting
  - Accounts payable/receivable automation
  - Compliance monitoring agent

### [GNUCash](https://github.com/Gnucash/gnucash)
- **License:** GPL-2.0
- **Language:** C / Scheme
- **Domain:** Personal/Small Business Accounting
- **Description:** Double-entry accounting with investments, budgeting, and financial reports. 25+ years of active development.

## Investment Research

### [OpenBB Platform](https://github.com/OpenBB-finance/OpenBB)
- **License:** AGPL-3.0
- **Language:** Python
- **Domain:** Investment Research Terminal
- **Description:** Open-source Bloomberg Terminal alternative. Stocks, crypto, forex, macro data. Plugin system for custom data sources. Built-in AI agent integration layer (v5.0).
- **AI Opportunities:**
  - Custom research agents on proprietary data
  - Automated earnings call summarization
  - Portfolio risk analysis with FinRL

## Trading & Market Data

### [Freqtrade](https://github.com/freqtrade/freqtrade)
- **License:** GPL-3.0
- **Language:** Python
- **Domain:** Algorithmic Crypto Trading
- **Description:** Full-featured trading bot with backtesting, paper trading, and live trading. ML-based strategy optimization via Hyperopt/Optuna.

### [FINOS Foundation](https://github.com/finos)
- **License:** Apache 2.0 (various)
- **Domain:** Financial Services Open Source
- **Description:** Fintech Open Source Foundation — maintains Legend (data modeling), FDC3 (desktop interoperability), and Symphony (secure messaging) under Apache 2.0.
