# Market Map — Financial AI (2026)

## Open Source Ecosystem Leaders

| Organization | Key Projects | Focus |
|-------------|-------------|-------|
| [AI4Finance Foundation](https://ai4finance.org/) | FinRL, FinGPT, FinRobot | Research-to-production AI for trading and analysis |
| [FINOS](https://github.com/finos) | Legend, FDC3, Symphony | Financial services open standards and interoperability |
| [Apache Foundation](https://apache.org/) | Fineract, OFBiz | Core banking and ERP infrastructure |
| [OpenBB Finance](https://github.com/OpenBB-finance) | OpenBB Platform | Investment research and data terminal |
| [Freqtrade](https://github.com/freqtrade) | Freqtrade | Algorithmic crypto trading |

## Competitive Landscape: Open vs Proprietary

```
Data Access:        Bloomberg/Refinitiv ←→ OpenBB + free APIs
Trading Execution:  Interactive Brokers  ←→ Freqtrade + broker APIs
Portfolio AI:       Kensho/Palantir      ←→ ai-hedge-fund + FinRL
LLM for Finance:    BloombergGPT         ←→ FinGPT + fine-tuned open models
Core Banking:       Temenos/Finastra     ←→ Apache Fineract
Accounting ERP:     SAP/Oracle           ←→ Apache OFBiz / Akaunting
```

## Key Globant Opportunity Zones

1. **AI-augmented core banking** — Fineract + LLM agents for loan decisioning and KYC
2. **Open-source investment research** — OpenBB + custom data plugins + FinGPT
3. **Multi-agent trading systems** — ai-hedge-fund architecture for asset managers
4. **Regulatory compliance automation** — FinGuard / LLM on regulatory documents
5. **SME financial automation** — Akaunting/iDURAR + AI invoice processing and forecasting
