# Market Map — Financial AI (2026)

| Organization | Key Projects | Focus |
|-------------|-------------|-------|

## Competitive Landscape: Open vs Proprietary

```
Data Access:        Bloomberg/Refinitiv ←→ OpenBB + free APIs
Trading Execution:  Interactive Brokers  ←→ Freqtrade + broker APIs
Portfolio AI:       Kensho/Palantir      ←→ ai-hedge-fund + FinRL
LLM for Finance:    BloombergGPT         ←→ FinGPT + fine-tuned open models
Core Banking:       Temenos/Finastra     ←→ Apache Fineract
Accounting ERP:     SAP/Oracle           ←→ Apache OFBiz / Akaunting
```

## LATAM-Specific Opportunities

| Vertical | Opportunity | Why Now |
|----------|-------------|---------|
| SMB lending | AI credit scoring on alternative data | 60-70% of LATAM SMBs lack formal credit access |
| Tax compliance | AI document agents for SPED/SAT/AFIP | Brazil, Mexico, Argentina tax complexity is high |
| Open banking | AI aggregation middleware | Brazil Open Finance Phase 4 live; Mexico following |
| Remittances | AI routing for lower fees | $150B+/year LATAM remittances; margin compression opportunity |
| AgriFintech | Crop insurance + inventory financing via AI | Satellite + AI data on agricultural assets |

Key LATAM players to know: Nubank (100M+ customers, full ML stack), Mercado Pago, PicPay, Clip (Mexico), Nequi (Colombia), Konfio, Kueski

## Key Globant Opportunity Zones

1. **Multi-agent trading infrastructure** — TradingAgents-style systems for proprietary trading desks (Apache-2.0, ready to customize)
2. **AI-augmented core banking** — Fineract + LLM agents for loan decisioning, KYC, fraud detection
3. **Open-source investment research** — OpenBB MCP + custom data plugins + FinGPT for wealth management
4. **Regulatory compliance automation** — FinGuard / RAG-on-regulatory-docs for RegTech clients
5. **LATAM SMB financial automation** — Akaunting/iDURAR + AI invoice processing and forecasting
6. **Open banking aggregation** — AI middleware connecting LATAM open banking APIs to enterprise dashboards
