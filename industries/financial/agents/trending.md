# Trending AI Agents — Financial (Week of 2026-07-02)

## Breakout Projects

### ai-hedge-fund (virattt/ai-hedge-fund) — 50K+ Stars
Crossed 50,000 GitHub stars, making it the most-starred open-source AI finance project. New in 2026:
- React Flow-based visual editor for building custom investment committees by drag-and-drop
- Real broker integration being added (Interactive Brokers API)
- 18-agent architecture now includes macroeconomic and geopolitical specialist agents
- Each agent produces structured reasoning with confidence scores

### FinGuard — New Compliance LLM Agent
New open-source compliance detection model initialized from Qwen3-8B, trained via supervised fine-tuning + self-play reinforcement learning:
- 90.23 / 85.43 F1 on compliance detection benchmarks
- Outperforms larger general-purpose models on FinReg tasks
- Detects financial regulatory non-compliance in LLM interactions
- [Paper: arxiv.org/pdf/2605.29427](https://arxiv.org/pdf/2605.29427)

### OpenBB Platform v5.0
- Native AI agent integration layer
- Plugin system for custom data providers
- Built-in support for agentic workflows over financial data

### FinRL RL-LLM Hybrid
- New paper: LLM as encoder for transaction embeddings fed to RL agent with business-aligned rewards
- Applied to fraud detection: publicly available on HuggingFace
- Shows 20-30% improvement over pure ML approaches on imbalanced fraud datasets
