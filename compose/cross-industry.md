# 🔗 Patrones cross-industry

> Recetas que aplican a múltiples industrias.

## El patrón universal Globant AI

```
1. Identificar vertical open source (Odoo, OpenMRS, Moodle, etc.)
2. Fork + customización de dominio
3. Capa agentica sobre el sistema base
4. UI conversacional / API
5. Deploy en infra del cliente o cloud
```
