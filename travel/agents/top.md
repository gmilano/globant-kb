## Primary Agents & MCP Servers

| Name | License | Stars | Description |
|------|---------|-------|-------------|
| [alibaba-flyai/flyai-skill](https://github.com/alibaba-flyai/flyai-skill) | MIT | ~150★ | Fliggy (Alibaba) full-stack travel MCP skill. The first production travel MCP skill from a major OTA. |
| [punitarani/fli](https://github.com/punitarani/fli) | MIT | ~2.4k★ | Google Flights MCP + CLI + Python library. Directly reverse-engineers Google Flights' internal API (no scraping). Works natively with Claude Code, Codex CLI, ChatGPT. TS port available as `fli-js` on npm. |
| [ravinahp/flights-mcp](https://github.com/ravinahp/flights-mcp) | MIT | ~205★ | MCP server for flight search via Duffel API. Supports complex multi-city searches with contextual memory for natural conversation. Plug into Claude Desktop in minutes. |
| [RollingGo-AI/rollinggo-hotel-mcp](https://github.com/RollingGo-AI/rollinggo-hotel-mcp) | MIT | ~80★ | Hotel booking MCP backed by DIDA (world's #3 B2B travel platform). 2M+ hotels, 110k+ direct-partner properties, real-time pricing. Free unlimited access. Hotel search, room types, pricing plans, cancellation policies. |
| [donghyun-chae/mcp-amadeus](https://github.com/donghyun-chae/mcp-amadeus) | MIT | ~60★ | Amadeus GDS MCP server. Connects AI agents to Amadeus Self-Service APIs: flight search, seat maps, airline information. Works with any MCP-compatible LLM. |
| [HarimxChoi/langgraph-travel-agent](https://github.com/HarimxChoi/langgraph-travel-agent) | MIT | ~45★ | Production-ready LangGraph multi-agent travel planner. Async parallel orchestration across Amadeus (flights), Hotelbeds (hotels), Twilio (SMS notifications), HubSpot (CRM). Best reference for enterprise travel deployments. |
| [extrawest/a2a_protocol_fundamentals_python](https://github.com/extrawest/a2a_protocol_fundamentals_python) | MIT | ~120★ | Multi-agent travel system via Google A2A protocol. Specialized agents for hotel booking (CrewAI), car rentals (LangGraph), currency conversion, and travel coordination — all communicating via A2A. |
| [naakaarafr/AI-Travel-Agent-Advanced](https://github.com/naakaarafr/AI-Travel-Agent-Advanced) | MIT | ~90★ | CrewAI + Gemini multi-agent trip planner. Destination analysis agent, local expert agent, itinerary agent. CLI interface producing 7-day plans with budgets, real-time web search for weather/flights/hotels/attractions. |
| [shaheennabi/Production-Ready-TripPlanner-Multi-AI-Agents-Project](https://github.com/shaheennabi/Production-Ready-TripPlanner-Multi-AI-Agents-Project) | MIT | ~74★ | Production-grade CrewAI trip planner. Discovers destinations, creates custom itineraries, finds local attractions, beach spots. Includes MLOps patterns and CI/CD pipeline examples. |
