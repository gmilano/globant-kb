## Globant Opportunity Map

### Near-Term (0-6 months)
1. **Corporate Travel Automation** — Amadeus SDK + policy enforcement. Entry: `travel-booking-agents` pattern.
2. **Hotel AI Concierge** — QloApps + Claude + WhatsApp. LATAM independent hotels.
3. **Tour Operator Digitization** — ExcursioX + AI. LATAM operators on WhatsApp + spreadsheets.

### Medium-Term (6-18 months)
4. **Agentic OTA Components** — Amadeus + Duffel SDK wrappers as MCP tools.
5. **Travel Data Infrastructure** — Clean data pipelines as foundation for client AI agents.
6. **NDC Integration Layer** — Help airlines/hotels connect to NDC + expose as agent APIs.

### Long-Term (18+ months)
7. **Consumer Agentic Booking** — Wait for trust gap to close (currently 2-7%, will be 20-30% by 2028).
