## LATAM Opportunity Map

### Why LATAM is a priority

| Factor | Detail |
|--------|--------|
| **Grid transition speed** | Brazil, Chile, Colombia adding 15-20 GW renewables/yr — grid integration AI needed NOW |
| **Underserved utilities** | Most LatAm utilities run on SAP + Excel — huge green-field for EMS AI |
| **EV surge** | Brazil EV sales +120% YoY (2025); smart charging AI under-deployed |
| **Energy poverty** | 20M+ households off-grid — AI-optimized microgrids offer leapfrog path |
| **Incumbent weakness** | Siemens/ABB/GE weak in smaller LatAm utilities; open-source AI+Globant = competitive |

## Globant Positioning

### Competitive Advantages
1. **Open-source first**: PyPSA + pandapower + OpenEMS + LangGraph — no license cost = faster client ROI
2. **LATAM presence**: on-the-ground in Brazil, Argentina, Colombia, Chile — client relationships with local utilities
3. **AI Studio depth**: LLM fine-tuning + agent orchestration capability beyond typical energy SIs
4. **Speed**: Globant can go from POC to production in 8-12 weeks vs 18+ months for incumbent OEMs

### Target client segments
- **Tier 2-3 utilities** in LATAM looking to modernize without SAP/GE/Siemens costs
- **Renewable IPPs** (solar/wind) needing AI forecasting and O&M optimization
- **Mining/industrial** companies with large energy bills needing AI demand response
- **EV charging networks** scaling fast and needing smart charging optimization

### Engagement models
- **AI accelerator (4 weeks)**: PyPSA + LangGraph POC on client grid data → show value, win larger engagement
- **EMS modernization**: OpenEMS/MyEMS deployment + LLM layer + operator chat (12-16 weeks)
- **Renewable forecasting service**: pvlib+windpowerlib+LSTM model → 30-day rolling forecast (8-10 weeks)
- **VPP AI platform**: Full DERMS agent on GridAPPS-D + OpenEMS (20-26 weeks)
